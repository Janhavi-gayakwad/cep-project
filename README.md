Sevatric is a web-based platform designed to provide users with easy access to various government schemes based on categories like education, agriculture, health, and more. The project aims to simplify information and make schemes accessible to students, farmers, and citizens.

🎯 Features
📚 Categorized government schemes (Education, Agriculture, Health, etc.)
👩‍🎓 Student-focused schemes (10th, 12th, scholarships)
🌾 Agriculture-related schemes
🏥 Health and welfare schemes
🧭 Easy navigation with multiple HTML pages
🖼️ Visual representation using images
🛠️ Tech Stack
HTML5 – Structure of the website
CSS3 – Styling and layout
JavaScript – Basic interactivity
Images/Web Assets – For better UI experience
